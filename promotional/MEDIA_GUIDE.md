# Media Guide

Place hotel photos under `promotional/public/media/` using the structure below.

## Main structure

- `public/media/site/`
- `public/media/hotel/`
- `public/media/rooms/junior-suite-king/`
- `public/media/rooms/junior-suite-twin/`
- `public/media/rooms/superior-king/`
- `public/media/rooms/deluxe-suite/`

## Recommended files

### Site-wide

- `public/media/site/rooms-hero.jpg`
  - Main hero image for the `/odalar` page.

### General hotel photos

- `public/media/hotel/hotel-exterior-01.jpg`
- `public/media/hotel/hotel-lobby-01.jpg`
- `public/media/hotel/hotel-breakfast-01.jpg`
- `public/media/hotel/hotel-detail-01.jpg`

These are reserved for future sections such as homepage, about, location, or contact.

### Room photos

Each room folder should contain:

- `cover.jpg`
  - Main image for room cards and the room detail hero.
- `gallery-1.jpg`
- `gallery-2.jpg`
- `gallery-3.jpg`
- `gallery-4.jpg`

## File rules

- Use only lowercase letters.
- Use hyphens, not spaces.
- Keep extension as `.jpg` for JPEG files.
- Prefer large source images. Recommended width is at least 1600px.
- Keep the cover image horizontally strong and visually clean.
- Gallery images can be mixed horizontal and vertical.
